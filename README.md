# ✨ Shayari Reader

A beautiful and simple **Shayari Reading Web App** where users can explore different categories of shayari like **Love, Sad, and Motivation**.  
The app provides a smooth reading experience with page animations, swipe navigation, and a favorite system.

## 📖 Features

✨ Browse Shayari by categories  
❤️ Add your favorite shayari  
📱 Mobile-friendly swipe navigation  
➡️ Next and previous shayari controls  
🎭 Smooth page transition animations  
📚 Shayari counter to track progress  
🎨 Clean and minimal UI design  

## 🗂️ Categories

- ❤️ Love Shayari
- 💔 Sad Shayari
- 🔥 Motivation Shayari

## 🚀 How It Works

1. Select a Shayari category.
2. Read beautiful Shayari one by one.
3. Swipe left/right or use navigation buttons to change pages.
4. Save your favorite Shayari.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
